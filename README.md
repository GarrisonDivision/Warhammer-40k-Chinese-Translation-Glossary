# Warhammer-40k-Chinese-Translation-Glossary
战锤 40K（Warhammer 40,000）中英术语翻译对照表

## 📖 项目说明
本仓库提供 **JSON 格式**的战锤40K标准译名对照表，面向：
- AI 大模型训练、翻译工具调用
- 中文小说翻译者、创作者
- 战锤玩家、设定爱好者

持续更新官方标准译名、社区通用译法、小说常用译法，统一中文翻译规范。
